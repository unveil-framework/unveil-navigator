# UT0702 - Obtain access to other source

**Tactic:** [UTA07 - Impact](./README.md)

## Description

The handler may target the source as a way of gaining access to a new source using connections to additional valuable sources within the same network or organization. This technique involves leveraging the initial source’s trust and access to introduce or recommend the handler (or another operative) to new contacts. This may occur through direct introductions, gaining entry to meetings, or being included in communications channels where the handler can establish relationships with these new sources.
