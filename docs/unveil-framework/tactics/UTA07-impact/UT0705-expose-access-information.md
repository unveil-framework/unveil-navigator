# UT0705 - Expose access information

**Tactic:** [UTA07 - Impact](./README.md)

## Description

The handler may aim to uncover specific details about the access levels or credentials that the source possesses within their network or organization. This technique focuses on obtaining information that can lead to gaining unauthorized access to restricted areas, systems, or data, crucial for the advancement of the operation. This might involve learning about security protocols, passwords, or physical access controls that the source has access to.
