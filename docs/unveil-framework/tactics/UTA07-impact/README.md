# UTA07 - Impact

## Description

The handler is trying to obtain the required information or access from the source in order to
fulfill the operation's requirements. Impact techniques regarding the collection of information
must not be confused with the information gathering of the screening phase, the first one being
the collection of specific information required by the agency while the second refers to finding
out general source areas of knowledge.
