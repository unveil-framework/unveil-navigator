# UT0705.001 - Expose access information: Security protocols

**Tactic:** [UTA07 - Impact](./README.md)

## Description

The handler may gather detailed information about the security protocols in place within the source’s organization. This includes information on cybersecurity measures, physical security procedures, and any emergency response protocols. Understanding these protocols allows the handler to plan how to circumvent security measures discreetly and safely, ensuring operational success while minimizing risks to the source and the operation.
