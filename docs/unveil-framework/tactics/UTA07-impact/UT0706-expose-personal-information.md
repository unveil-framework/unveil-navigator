# UT0706 - Expose personal information

**Tactic:** [UTA07 - Impact](./README.md)

## Description

The handler may aim to use the source to expose personal information to gain deeper psychological leverage or compliance. This includes personal details about the source (such as personal beliefs, family matters, financial information, or health issues amongst others) which may have been identified as an intelligence requirement. The main difference with any control technique involving personal information is that the ultimate goal of UT0706 - Expose personal information is the exposure of the information itself.
