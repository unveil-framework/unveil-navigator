# UT0104 - Discern termination criteria

**Tactic:** [UTA01 - Plan](./README.md)

## Description

The handler may need to define under which criteria a HUMINT operation should be terminated. Termination criteria for a HUMINT operation are vital to ensure that resources are not wasted and that operational integrity is maintained. These criteria help determine when an operation should be concluded, either due to the fulfillment of its objectives, the reduction of benefits from continuing the operation or other strategic reasons. Identifying specific potential termination criteria at the planning phase can save time from being wasted in later phases. This technique is related with the execution of those in UTA09 - Exit, thus this being part of the planning of said phase.
