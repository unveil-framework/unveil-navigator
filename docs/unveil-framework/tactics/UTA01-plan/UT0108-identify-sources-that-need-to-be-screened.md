# UT0108 - Identify sources that need to be screened

**Tactic:** [UTA01 - Plan](./README.md)

## Description

The handler may identify sources that need to be screened as part of the planning phase of the HUMINT operation. Screening refers to the process of obtaining information about them in order to evaluate and select those that could be a better match regarding the operation requirements. This process involves identifying both the direct source, the individual who has access to the information required by the handler, and the different indirect or secondary sources, which are individuals that have access to the direct source. Indirect sources can be approached if the handler assesses that the direct source can not be approached because it would pose a high risk for them, the operation or the agency.
