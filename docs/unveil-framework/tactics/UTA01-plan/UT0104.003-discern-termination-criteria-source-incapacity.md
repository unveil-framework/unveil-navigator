# UT0104.003 - Discern termination criteria: Source incapacity

**Tactic:** [UTA01 - Plan](./README.md)

## Description

The handler may terminate the operation if the source is no longer capable of providing information due to incapacitation. This could be due to physical, psychological or social factors that prevent the source from further participation in the operation. Recognizing these signs early helps in time-saving efforts in the pursuit of the operation’s objectives.
