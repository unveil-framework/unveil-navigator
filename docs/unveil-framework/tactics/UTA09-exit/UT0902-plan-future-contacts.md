# UT0902 - Plan future contacts

**Tactic:** [UTA09 - Exit](./README.md)

## Description

The handler may finish a contact or interaction with a source by laying the groundwork for future contacts. This might involve scheduling future meetings, suggesting topics for future discussion, or other pretexts that justify continued engagement. This method helps maintain the relationship’s viability and ensures ongoing access to valuable intelligence. The difference with “UT0901 - Exit contact” is that, in this technique, the closure of the contact maintains an agenda or list of scheduled tasks or interests. This technique can only be used with formal contacts because these are defined as the only sources with whom future encounters can be planned. Moreover, if used subtly, this technique can be also used with continuous contacts in elicitation operations, as long as the source does not know who they are collaborating with.
