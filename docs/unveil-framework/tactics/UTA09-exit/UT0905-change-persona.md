# UT0905 - Change persona

**Tactic:** [UTA09 - Exit](./README.md)

## Description

The handler may introduce the source to another fake virtual persona operated by another handler. This technique is used to transition the source smoothly from one handler to another without disrupting the flow of information or, even, alerting the source to the change. In this technique, the source does not know they are interacting with an intelligence agency. Note that any handler selection techniques seen in previous sections apply here.
