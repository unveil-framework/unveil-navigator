# UT0906 - Ghosting

**Tactic:** [UTA09 - Exit](./README.md)

## Description

The handler may stop answering the source’s messages either progressively or abruptly to reduce or cut contact, ensuring a non-confrontational withdrawal from communication. Note that this may have emotional implications for the source if unexplained, but it can be a tool to face situations in which the handler or the source may be overinvolved.
