# UT0602.007 - Use incentives: Continuous program

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may employ a continuous reinforcement schedule, where a reinforcement is provided every time the desired behavior is exhibited by the source. Continuous reinforcement ensures a consistent association between the behavior and the reinforcement, which can be effective in establishing trust and encouraging desired behaviors. However, this approach may lead to dependency on the reinforcement and may not be sustainable in the long term. Thus, a continuous program is recommended at the start of the reinforcement of a response in order to quickly establish it, but it is recommended to subsequently change to an intermittent program to ensure the behavior remains. Moreover, the program must transition to natural reinforcements (those reinforcements that can be spontaneously found in the source’s daily life) in order for the behavior to be maintained even without the handler’s involvement.
