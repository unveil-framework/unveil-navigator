# UT0601 - Control the conversation

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may try to gain control of the collection effort during the conversation, which is achieved when the source is answering the questions the handler poses. Once control has been achieved it must be maintained or regained if necessary. This involves a deliberately implemented operational approach to conversation management, where the handler needs to be adept in both listening and guiding the dialogue to extract valuable intelligence without alerting the source.
