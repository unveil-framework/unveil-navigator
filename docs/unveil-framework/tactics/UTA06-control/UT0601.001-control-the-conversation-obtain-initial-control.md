# UT0601.001 - Control the conversation: Obtain initial control

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may try to obtain control of the conversation. US's generally accepted doctrine in interrogations involves using open-ended questions that encourage the source to talk more extensively, thereby providing more information and opportunities for follow-up and avoiding yes/no questions. Establishing this dynamic early on is crucial as it sets the tone for the interaction, making it easier for the handler to guide the conversation effectively. Moreover, innocuous or seemingly unrelated questions to the topic of interest might be used in order to keep the source from developing suspicions.
