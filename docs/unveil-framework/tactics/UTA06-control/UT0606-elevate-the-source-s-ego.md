# UT0606 - Elevate the source's ego

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may elevate the ego of a low self-esteem source or an egocentric source by flattering them in order to make them interact more. This can be done by asking about what the source does best, showing interest in them or praising them into admitting their role in an event, amongst others.
