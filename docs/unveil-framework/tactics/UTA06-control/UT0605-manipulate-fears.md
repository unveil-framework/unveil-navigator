# UT0605 - Manipulate fears

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may link existing or newly created feelings of fear towards an object to its reduction to manipulate the source into collaborating. This technique requires the use of positive or negative reinforcement techniques in the tactic UTA0005 - Influence. Moreover, the specific implementation of this technique can be consulted in the technique “UT0512.008 Argument in itself: Fear”. In an online HUMINT context, a handler could manipulate a source by amplifying their fear of a specific cybersecurity threat, such as a data breach. The handler might first spread targeted information about how widespread and devastating this threat could be, creating a heightened sense of fear and urgency in the source. Once the source is sufficiently alarmed, the handler would then offer "solutions" or guidance that appears to mitigate this fear—such as exclusive access to protective measures or insider information on avoiding breaches.
