# UT0602.008 - Use incentives: Intermittent program

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may implement an intermittent reinforcement schedule, where rewards or reinforcements are provided only occasionally or unpredictably following the display of desired behaviors by the source. Intermittent reinforcement can be more resistant to extinction and can maintain desired behaviors over a more extended period. This technique involves varying the timing, frequency or magnitude of reinforcements to keep the source motivated and engaged in displaying trusting behaviors. However, a continuous program is recommended at the start of the reinforcement of a response in order to quickly establish it, but it is recommended to subsequently change to an intermittent program to ensure the behavior remains. Moreover, the program must transition to natural reinforcements (those reinforcements that can be spontaneously found in the source’s daily life) in order for the behavior to be maintained even without the handler’s involvement.
