# UT0611 - Seduce the source

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler tries to seduce the source in order to lower their defenses and make them cooperate either knowingly or unknowingly. This subtechnique is dangerous due to the possibility of the handler developing feelings for the source, which should be prevented at all costs. It must be noted that, in order to correctly seduce a source, they should be previously screened because the approach must be tailored in detail.
