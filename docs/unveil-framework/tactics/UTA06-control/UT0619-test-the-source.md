# UT0619 - Test the source

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may give false information to the source in order to check if the person is trustworthy or if they give the information to someone else. The difference between this technique and the Impact technique “VT0707 - Deceive the adversary” is that, in UT0619, the handler's goal is to check if their control over the source is intact and if they can continue working with them, whereas the goal of VT0707 is to feed false information to an adversary via the source.
