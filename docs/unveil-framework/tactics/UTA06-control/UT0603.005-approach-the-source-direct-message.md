# UT0603.005 - Approach the source: Direct message

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may send a direct message to the source through social media platforms such as Instagram or Twitter, underground forums such as Exploit.in or Breachforums, or through any other similar platform. Independently of the pretext used by the handler, this subtechnique focuses on direct contacts with the source initiated by a handler through an online platform.
