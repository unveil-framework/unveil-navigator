# UT0610 - Establish an equal status

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler establishes a relationship in which the source thinks both parts have an equal status. For example, this can be done by adopting a persona of the same rank or group as the source. Establishing a relationship where the source perceives both parties as having equal status can be highly effective, especially in contexts where trust and mutual respect are crucial for information exchange. This technique is useful when the handler aims to gain the source's confidence and access to sensitive information or insider knowledge without raising suspicion.
