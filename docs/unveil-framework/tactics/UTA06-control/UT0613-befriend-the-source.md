# UT0613 - Befriend the source

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may try to develop a friendship-kind of relationship with the source in order for them to collaborate and reveal information. The handler should not get personally involved with the source and remain emotionally neutral at all costs, even when displaying friendshiplike interactions. This technique can be used as a previous step to UT061 - Seduce the source.
