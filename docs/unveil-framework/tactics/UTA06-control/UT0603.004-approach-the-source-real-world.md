# UT0603.004 - Approach the source: Real world

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler exploits a situation in the physical environment of the source that implies direct interaction. For example, the handler might arrange to 'accidentally' meet the source at an event both are likely to attend or create a scenario where the source can play a hero, victim, beneficiary or other. This method helps to establish a connection in a natural setting, potentially leading to a more robust relationship based on shared experiences.
