# UT0603.003 - Approach the source: Pretext

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler uses a carefully constructed scenario or role that serves as a cover for their true intentions. This could involve posing as a journalist, researcher, developer, affiliate or any other plausible identity that facilitates engagement without raising suspicions. The pretext approach allows the handler to gather information under the guise of legitimate activities, making it easier to access sensitive or guarded information without alerting the source.
