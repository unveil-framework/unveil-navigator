# UT0620 - Establish security words

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may agree with the source on the use of security words or phrases easily identifiable by both parties so as to let the other know that he/she is not in a condition to speak freely. This can be used in either messaging apps or voice calls indistinctly. Although several words for different situations can be agreed, it is recommended to assess to what extent the source will be capable of remembering the keys when needed, specially, considering that these words will not be used on a regular basis in standard interactions but on situations of stress or risk for the source.
