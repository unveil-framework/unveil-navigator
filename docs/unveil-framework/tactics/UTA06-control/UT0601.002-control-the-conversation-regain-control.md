# UT0601.002 - Control the conversation: Regain control

**Tactic:** [UTA06 - Control](./README.md)

## Description

The handler may try to regain control of the conversation if it seems to be leaving the desired course by employing techniques such as redirecting questions, using affirmations that reinforce the source’s cooperation or subtly shifting the topic back to the main agenda. This technique also includes any interaction in which the handler makes sure that they are obtaining more information than they are giving, for example, by answering questions with other questions. This is particularly important in maintaining the focus of the conversation on the intelligence objectives and preventing the source from dominating the interaction or steering it toward less relevant or sensitive topics. In elicitation operations, it is crucial for the handler to redirect the conversation without the source noticing or being alerted.
