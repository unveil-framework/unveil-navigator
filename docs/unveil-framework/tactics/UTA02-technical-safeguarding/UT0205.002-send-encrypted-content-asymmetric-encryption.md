# UT0205.002 - Send encrypted content: Asymmetric encryption

**Tactic:** [UTA02 - Technical Safeguarding](./README.md)

## Description

The handler may use asymmetric encryption (e. g., PGP) to send content. In asymmetric encryption two different keys are required for encryption and decryption. This technique is vital for secure communication over unsecured networks, where key exchange might otherwise compromise the encryption.
