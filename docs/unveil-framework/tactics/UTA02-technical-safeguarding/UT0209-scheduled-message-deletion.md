# UT0209 - Scheduled message deletion

**Tactic:** [UTA02 - Technical Safeguarding](./README.md)

## Description

The handler may set up communications to auto-delete based on a pre-set schedule, ensuring that sensitive information does not remain accessible beyond its useful life. Note that deleting messages from a device may harden the way in which this information is obtained later and will not be as reachable as expected under certain circumstances.
