# UT0214.002 - Use cryptocurrencies: Non-KYC exchanges

**Tactic:** [UTA02 - Technical Safeguarding](./README.md)

## Description

In contrast to KYC exchanges, non-KYC exchanges do not require users to provide personal identification information. These exchanges allow the handler to maintain a higher degree of anonymity when conducting cryptocurrency transactions. Non-KYC exchanges are often preferred in operations where maintaining operational security and avoiding linkages to realworld identities is critical. The handler can use these platforms to acquire, trade, or convert cryptocurrencies without leaving a traceable identity behind. However, these exchanges may have limitations on transaction volumes or the range of supported currencies, which must be considered during operational planning.
