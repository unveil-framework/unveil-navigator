# UT0204 - Use file encryption

**Tactic:** [UTA02 - Technical Safeguarding](./README.md)

## Description

The handler may implement file-specific encryption, opting to encrypt individual files rather than the entire operating system. This targeted encryption allows for greater control over which data needs higher levels of security, reducing resource usage and enhancing access to nonsensitive data. This is different from UT0403 - Use encrypted devices because this technique was focused on the encryption of the whole device while UT0404 - Use file encryption is focused on the encryption of specific files or information.
