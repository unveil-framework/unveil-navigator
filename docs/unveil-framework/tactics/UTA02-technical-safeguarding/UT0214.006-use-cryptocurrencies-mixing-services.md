# UT0214.006 - Use cryptocurrencies: Mixing services

**Tactic:** [UTA02 - Technical Safeguarding](./README.md)

## Description

The handler may use cryptocurrency mixing services to further obscure the origins and destinations of cryptocurrency transactions. Mixing services scramble operations conducted by multiple users, breaking the clear trace of funds from sender to receiver. This method disperses transaction details across a wide array of unrelated transactions, making it incredibly difficult for observers to trace the funds' original source or eventual destination. By using mixing services, the handler adds an additional layer of anonymity to cryptocurrency transactions, enhancing the confidentiality and security of financial operations critical to the mission’s success.
