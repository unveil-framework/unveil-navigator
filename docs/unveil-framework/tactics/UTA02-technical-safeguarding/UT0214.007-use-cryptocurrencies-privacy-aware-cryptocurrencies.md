# UT0214.007 - Use cryptocurrencies: Privacy aware cryptocurrencies

**Tactic:** [UTA02 - Technical Safeguarding](./README.md)

## Description

The handler may opt to use privacy-focused cryptocurrencies, such as Monero 76, Zcash and Dash, which offer built-in anonymity features that obscure transaction details to a greater extent than more transparent blockchain technologies like Bitcoin 77. These cryptocurrencies use various methods to ensure that all (in the case of Monero78) or part of the transactions (in the case of Zcash and Dash) are not traceable and remain confidential. By leveraging these privacy-aware cryptocurrencies, the handler ensures that financial transactions related to the operation leave minimal digital footprints, significantly reducing the risk of exposure and tracking by adversaries.
