# UT0301.012 - Profile the possible sources: IQ

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may find out the source's Intelligence Quotient (IQ), which can help tailor the approach techniques and collection process. Note that this technique explicitly refers to the IQ as the standardized score used to measure human intelligence and cognitive abilities relative Sociales 13, no. 2 (1974): 65–93, https://doi.org/10.1177/053901847401300204.

to a statistically normal distribution described by the American Psychology Association, not to the knowledge or intelligence value that a source may have within a given operation. The source’s IQ may also be considered when identifying the level of risk the source poses to the handler and the operation.
