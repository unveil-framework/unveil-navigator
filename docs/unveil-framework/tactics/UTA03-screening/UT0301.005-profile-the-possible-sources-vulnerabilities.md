# UT0301.005 - Profile the possible sources: Vulnerabilities

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may find out the source's vulnerabilities in order to subsequently exploit them during the collection process. Although different ways to categorize vulnerabilities in human intelligence have been proposed, such as the MICES model (money, ideology, coercion, ego or sex), in this context a vulnerability should be understood in a wider range. Thus, a vulnerability can be anything that can be exploited by the handler in order to obtain information from a source, and it can be related to other factors detected in the screening process (relationships, social identity, values, personality…).
