# UT0302.001 - Choose a source: Assess willingness to cooperate/Hostility

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may evaluate the source's willingness to cooperate and degree of hostility. This can be done in multiple ways, by asking general questions to see if they answer or hesitate to do so, or by indirectly assessing this based on the profile's information, for example.
