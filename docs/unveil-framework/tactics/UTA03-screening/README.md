# UTA03 - Screening

## Description

The handler is trying to obtain information about the possible sources in order to evaluate them
and select those that could be a better match regarding the operation requirements. This tactic
includes actions regarding the profiling of the sources and the identification of their baseline if
possible, as well as the selection of one or more sources. Though in some manuals the
concept of screening is exclusively used to refer to the process of identifying a source's access
to the information and willingness to collaborate, in this model it also includes the profiling of a
source, the identification of their baseline and the action of choosing an individual.
Although the process of screening sources can be done in parallel with the collection of
information, it is advised to be done previously in order to both it be done adequately and to
subsequently tailor the strategy to the chosen source. Moreover, the screening process may
be carried out by a professional other than the final handler, namely the screener or profiler,
which will in turn subsequently pass the profile information onto the handler or the handler's
team.
