# UT0303 - Identify source baseline

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may identify the source's communication baseline in order to later identify changes that may indicate a shift in the source’s mood, situation or veracity . In exclusively online relationships, baseline changes may also be an indicator of a change in the person behind a profile, if said profile is managed by more than one individual.
