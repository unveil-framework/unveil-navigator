# UT0301 - Profile the possible sources

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may profile the different possible sources, that is, to conduct a systematic gathering and analysis of information about said individual. The profiler's goal is to understand their backgrounds, personality, behaviors, motivations and potential vulnerabilities. This technique may be conducted by the same handler that will approach the source or by a selected professional with profiling skills.
