# UT0301.026 - Profile the possible sources: Perceptual filters (PALIO)

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may find out the source’s most pervasive perceptual filter. This technique involves identifying which type of content the source most commonly directs their attention to. The PALIO filter classifies these contents into people, actions, places, information or objects. Understanding the source’s primary perceptual filter can help in tailoring messages and interactions to match the source’s natural focus of attention.
