# UT0301.027 - Profile the possible sources: Sensory filters (VAKO)

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may find out the source’s most predominant sensory filter. This involves identifying which sensory modality—visual, auditory, kinesthetic or olfactory—the source primarily uses to process information. Understanding the source’s sensory filter can aid in communicating more effectively by aligning with the source’s preferred mode of receiving information.
