# UT0301.028 - Profile the possible sources: Processing size

**Tactic:** [UTA03 - Screening](./README.md)

## Description

The handler may find out the source’s preferred processing size, that is, whether they tend to process information in big or small chunks. People who process information in big chunks may prefer a holistic view of situations, while those who process in small chunks may prefer detailed, step-by-step information. Profiling the source’s processing size can help in structuring communications in a way that the source finds most comprehensible and engaging.
