# UT0801.002 - Report source's personal information: Personality profile

**Tactic:** [UTA08 - Debrief and Archive](./README.md)

## Description

The handler documents the personality features of the source, behavioral trends, psychological profiles, and any other predictable behavior that could be used in the context of the relationship. In order to be able to effectively understand the interactions, it is recommended to use a tested psychological model such as the Eysenck’s model or any other well-known and standardized model within the organization so as to ease the comprehension and transference of information between handlers if needed.
