# UT0801.004 - Report source's personal information: Affiliations

**Tactic:** [UTA08 - Debrief and Archive](./README.md)

## Description

The handler documents any affiliations the source may have within political spheres, trade unions, activist organizations, religious groups, and other significant social or cultural entities. This information should be attached to the profile to be taken into account to deeply understand the source's network, influences, and potential biases while anticipating any potential conflict of interests.
