# UT0807 - Non-standard multimedia recording

**Tactic:** [UTA08 - Debrief and Archive](./README.md)

## Description

The handler may make efforts to register content which is not expected to be stored by default in the tools used employing third party applications. This can involve the storage of photographs which are shared only once by using third party applications to grab the information or recording conversations or messages without explicitly notifying the target.
