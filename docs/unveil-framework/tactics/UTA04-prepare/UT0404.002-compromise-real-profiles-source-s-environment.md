# UT0404.002 - Compromise real profiles: Source’s environment

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler may compromise the real profiles of the source's environment, including individuals, organizations, or entities closely associated with the source. This could involve targeting friends, family members, colleagues, or employers of the source in order to gain access to information indirectly related to the source. By infiltrating the source's social circle or professional network, the handler can gather additional intelligence and leverage these connections to manipulate or influence the source.
