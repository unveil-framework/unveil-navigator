# UT0409.001 - Prepare the approach: Develop a pretext

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler develops a carefully constructed scenario or role that serves as a cover for their true intentions. This could involve posing as a journalist, researcher, developer, affiliate, or any other plausible identity that facilitates engagement without raising suspicions. The pretext approach allows the handler to gather information under the guise of legitimate activities, making it easier to access sensitive or guarded information without alerting the source. The chosen pretext must be coherent with the control, obtaining trust and persuasion techniques that will subsequently be used with the source.
