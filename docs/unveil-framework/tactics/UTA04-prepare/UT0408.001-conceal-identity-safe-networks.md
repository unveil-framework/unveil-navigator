# UT0408.001 - Conceal identity: Safe networks

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler may use safe networks to conceal their identity and activities from detection. This entails operating within secure network environments that are resistant to surveillance and monitoring. By leveraging virtual private networks (VPNs), proxies, secure servers, and other network security measures, the handler can maintain anonymity and protect the confidentiality of their interactions with the source.
