# UT0404 - Compromise real profiles

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler may compromise profiles in order to gain access to information from parties somewhat related with the source. Compromise can take place using several methods that may include (but are not limited to) phishing, spear phishing, vulnerabilities exploitation, watering-hole attacks amongst others. This action may have serious legal and ethical implications, as well as major reputational implications in case of being discovered.
