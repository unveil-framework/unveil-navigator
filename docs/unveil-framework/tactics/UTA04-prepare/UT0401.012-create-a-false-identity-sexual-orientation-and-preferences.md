# UT0401.012 - Create a false identity: Sexual orientation and preferences

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler may enrich a previously designed or a newly created identity by including their sexual orientation (heterosexual, homosexual, bisexual, asexual or any other) as well as their sexual preferences, including kinks and paraphilias, in order for them to be considered in the planning of the strategy process, especially in approaches that include seducing the source.
