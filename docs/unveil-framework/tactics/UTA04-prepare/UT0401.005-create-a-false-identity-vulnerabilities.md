# UT0401.005 - Create a false identity: Vulnerabilities

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler may use a previously designed or a newly created identity with a specific set of vulnerabilities that must be correlated and coherent with the rest of the profile (relationships, social identity, values, personality, etc.). Vulnerabilities can be a useful tool to build rapport if used accordingly. In this context, a vulnerability is anything that could be exploited by an experienced individual in order to manipulate the persona.
