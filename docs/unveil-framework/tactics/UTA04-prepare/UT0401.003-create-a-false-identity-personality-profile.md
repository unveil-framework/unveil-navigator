# UT0401.003 - Create a false identity: Personality profile

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler may use a previously designed or a newly created identity with a specific personality profile according to the personality model previously followed in the screening model. It must be noted that the selected handler or handlers must be capable of portraying the designed profile, including the display of the indicators of the selected traits. The personality profile must be coherent with the rest of information detailed in the false identity, including relationships, geographical profile, susceptibility to incentives, etc.
