# UT0407.004 - Develop a baseline: Number of messages

**Tactic:** [UTA04 - Prepare](./README.md)

## Description

The handler may establish a baseline for the frequency and volume of messages exchanged with the source. This involves monitoring and controlling the rate at which messages are sent and received to maintain a consistent level of engagement over time, as well as the length of the message and the latence, that is, the time that it takes the fake persona to write a new message. By adhering to a predetermined number of messages per interaction session or per day, the handler can regulate the flow of communication and avoid raising suspicions about the authenticity of the persona or the nature of the relationship with the source.
