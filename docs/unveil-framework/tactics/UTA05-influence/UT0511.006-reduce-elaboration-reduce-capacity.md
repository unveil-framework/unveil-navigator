# UT0511.006 - Reduce elaboration: Reduce capacity

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may reduce the source's capacity to process by using several variables such as incrementing distractions, no repetition of the message, speaking fast so that the source can’t fully understand the message, using an oral channel (voice over text), etc. The reduction of the capacity is related to any action conducted by the handler to reduce the processing capacity of the target. This can include, but are not limited to, any kind of instigation to consume substances of any kind or instigation of sexual behavior to deliberately mine the processing capacity of the target. Other possible ways to reduce processing capacity include tiring the source or introducing distractions.
