# UT0511 - Reduce elaboration

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may use different variables in order to reduce the elaboration of a receptor that initially appears to be in the middle of the elaboration spectrum to subsequently use techniques that enhance persuasion in individuals with a low level of elaboration 111. This technique must be used before presenting the message to the source.
