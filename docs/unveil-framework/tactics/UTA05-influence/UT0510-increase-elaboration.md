# UT0510 - Increase elaboration

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may use different variables in order to increase the elaboration of a receptor that initially appears to be in the middle of the elaboration spectrum, to subsequently use techniques that enhance persuasion in individuals with a high level of elaboration 110. This technique must be used before presenting the message to the source.
