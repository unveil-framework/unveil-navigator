# UT0514 - Metacognition and self-validation

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may persuade a source in the higher side of the elaboration spectrum by making them validate their previously formed thoughts or attitudes regarding the persuasive object. The variables used in this technique must be presented to the source after the exposition of arguments114. It must be noted that, when using this technique, the source will validate their own existing thoughts about a proposal or object, whether they are positive or negative, so it is key for the handler to previously make sure that the source's thoughts are the ones they want them to have. For example, if the attitude the handler is trying to change is how a source feels about an adversary, then they must ensure their thoughts towards that adversary are negative prior to using metacognition techniques.
