# UT0510.011 - Increase elaboration: Increase capacity

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may affect the source's capacity to process, that is, enhancing it, by using several variables such as reducing distractions, moderate repetition of the message, speaking slowly so that the source can fully understand the message, using a written channel (text over voice), etc.
