# UT0501 - Tailor the message

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may design or adapt the messages given to the source so that they match with their characteristics, which can be used to enhance persuasion and trust. This technique relies on the persuasion technique of tailoring, a specific type of matching which relates the characteristics of the receptor (the source, in HUMINT contexts) to the variables used in the persuasive message in order to enhance persuasion87 88. In this sense, the personality of the source, as well as other psychosocial, perceptual and cognitive characteristics will be taken into account when communicating with the source.
