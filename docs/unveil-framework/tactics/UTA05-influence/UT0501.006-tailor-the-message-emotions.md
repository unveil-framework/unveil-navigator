# UT0501.006 - Tailor the message:Emotions

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may tailor their messages to the source's emotional valence, either positive or neutral emotions, or to a specific emotion 97. Therefore, by using messages that refer to the same emotion, or just the same valence, as the source's at that particular moment may enhance trust in the handler.

For example, in chat conversations, this might be done by adapting the frequency and variety of emoticons, gifs, stickers and memes to the ones the source uses during their communication. In this way, if a source seems to have a positive emotion, the handler can react with text messages or multimedia, such as emoticons, that display a positive emotion, or even the same as the source’s.
