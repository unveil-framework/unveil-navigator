# UT0504.002 - Use information probes: Categorical statement

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may use categorical statements to elicit reactions from the source. By making a definitive statement about a topic, the handler can prompt the source to either confirm, deny or correct the information, thus revealing their true knowledge or opinions. This technique relies on the source's natural inclination to respond to strong assertions, providing the handler with valuable data without appearing overly inquisitive.
