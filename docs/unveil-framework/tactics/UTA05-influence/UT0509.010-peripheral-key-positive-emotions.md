# UT0509.010 - Peripheral key: Positive emotions

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may evoke positive emotions to enhance persuasion. Using language, imagery, or storytelling that elicits feelings of joy, hope or excitement can create a favorable emotional state in the source. Leveraging positive emotions as a peripheral key enhances the source's receptiveness to the handler's persuasive messages.
