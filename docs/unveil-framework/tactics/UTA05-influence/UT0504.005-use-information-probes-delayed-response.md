# UT0504.005 - Use information probes: Delayed response

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may intentionally delay the response to tempt the source to reveal additional information. This technique can take place in live conversations by means of deliberated silences, or in text conversations by intentionally delaying a response for minutes, hours, or more. While in live conversations nonverbal cues can provide additional information, note that how busy the source is can lead to misinterpretations, making it specially relevant to obtain information about whether the source is in a context in which it cannot give an immediate answer for legit reasons, such as working, resting or staying in social events. In this regard, parallel publications in other social networks or activity in known used apps, amongst others can help to assess if the delay can be explained as an intentional delayed response, a lack of interest or others.
