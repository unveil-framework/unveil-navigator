# UT0504.003 - Use information probes: Intentional misstatement

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may intentionally make incorrect statements to observe the source's response. This technique involves deliberately providing false or misleading information to see if the source corrects the handler or provides additional details. The source's corrections or elaborations can reveal their level of knowledge and perspective on the subject, aiding the handler in understanding the source's true position and information.
