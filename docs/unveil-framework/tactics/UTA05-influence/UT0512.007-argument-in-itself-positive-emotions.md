# UT0512.007 - Argument in itself: Positive emotions

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may evoke positive emotions by associating the proposal with feelings of happiness, satisfaction, or excitement. This can be achieved through compelling storytelling, inspirational messages, or appealing to the source's values and aspirations. By eliciting positive emotions, the handler can create a favorable emotional state that enhances the source's receptiveness to the proposal and facilitates persuasion, but only when the source does not think that thinking more about the object of persuasion will decrease their positive emotions.
