# UT0510.009 - Increase elaboration: Rhetorical questions

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may use rhetorical questions to stimulate the source's cognitive engagement if the source is poorly motivated to think about the message. Rhetorical questions do not require a direct answer, but rather encourage the source to think more deeply about the topic. This technique can spark curiosity, prompt internal reflection and foster a more active processing stance, leading to enhanced elaboration of the information presented.
