# UT0504.001 - Use information probes: Joke or sarcastic comment

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may use jokes or sarcastic comments as a way to probe for information. By making a light-hearted or seemingly offhand remark, the handler can gauge the source's reactions and responses to sensitive topics without direct questioning. This technique helps the handler to gather insights in a non-threatening manner, often leading the source to reveal information they might otherwise withhold in a more serious context.
