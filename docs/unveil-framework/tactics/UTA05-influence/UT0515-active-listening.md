# UT0515 - Active listening

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may show they are actively listening to make the source feel more understood, making them develop a higher trust on the handler. This technique can be easily generated by avoiding interrupting, actively acting as if the handler is listening (nodding in video communication, sending little messages such as “oh, I get it” while the source is writing via text), asking clarifying questions, etc.
