# UT0501.011 - Tailor the message:Political affiliation

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may tailor their messages to resonate with the source's political affiliation or ideological stance. By addressing issues or topics in a manner consistent with the source's political beliefs, the handler can establish common ground and credibility. For example, if the source is known to have conservative views, the handler could discuss topics like national security, economic policies or patriotism in a way that resonates with conservative values.
