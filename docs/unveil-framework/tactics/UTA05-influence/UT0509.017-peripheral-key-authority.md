# UT0509.017 - Peripheral key: Authority

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may try to represent a figure of authority in order to use it as a peripheral key that persuades a source in low elaboration. However, it must be noted that authority may not have the same effect on every source, so it is highly relevant to understand how each individual feels about it, as authority figures may trigger trauma, generate rejection, anger or other negative feelings for some people, while for others it can be attractive.
