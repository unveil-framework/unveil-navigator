# UT0504 - Use information probes

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may use information probes to subtly gather information from the source without arousing suspicion. These probes are designed to elicit responses that reveal useful intelligence while keeping the source unaware of the handler's true objectives. Probes can take various forms, such as jokes, sarcastic comments, categorical statements, intentional misstatements, or changing the conversation or situation conditions. The handler must be skilled in interpreting the responses to these probes to gain insights while maintaining control over the interaction.
