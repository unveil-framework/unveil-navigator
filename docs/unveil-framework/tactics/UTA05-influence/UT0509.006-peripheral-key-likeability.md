# UT0509.006 - Peripheral key: Likeability

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may cultivate likeability or sympathy to enhance their persuasive influence on the source. Demonstrating warmth, empathy, and genuine concern for the source's well-being can foster positive feelings and rapport. By eliciting likeability or sympathy, which can be done by using compliments, the handler can increase the source's receptiveness to their persuasion attempts.
