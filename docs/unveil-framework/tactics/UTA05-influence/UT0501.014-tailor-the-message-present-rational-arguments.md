# UT0501.014 - Tailor the message:Present rational arguments

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may tailor their messages to present rational arguments and logical reasoning that appeal to the source's analytical thinking. Some personality profiles might be more easily persuaded by rational arguments (low neuroticism individuals102, for example), although vulnerability to these kinds of arguments will also depend on if the attitude we are trying to change is based on emotion or cognition. Rational arguments seem to be as persuasive as emotional arguments when the attitude has a cognitive base, but they do not seem to be as persuasive on emotionally-based attitudes103.
