# UT0518 - Anchoring

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The source may use conversation anchors to establish a reference around which the source's thoughts will then be based. In the context of persuasion and influence, an anchor is a reference point or initial piece of information that sets the standard or baseline for subsequent judgments and decisions. For example, in negotiations, the initial offer serves as an anchor, influencing how the rest of the discussion unfolds and what is perceived as a reasonable outcome.
