# UT0509.013 - Peripheral key: Affirmative phrases

**Tactic:** [UTA05 - Influence](./README.md)

## Description

The handler may use affirmative phrases to enhance persuasion. In this sense, sources on low elaboration seem to be better persuaded when the message is formulated as an affirmative phrase (“this is a good idea”) than when using negative formulas (“this is not a good idea”).
