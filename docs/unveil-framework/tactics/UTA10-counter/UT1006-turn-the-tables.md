# UT1006 - Turn the tables

**Tactic:** [UTA10 - Counter](./README.md)

## Description

The handler may use the source's suspicious behavior to their advantage by turning the situation around. This could involve leveraging the detected behavior to extract additional information, gain leverage or manipulate the source into revealing more than they intended. This technique requires careful handling to avoid arousing suspicion or damaging the relationship with the source.
