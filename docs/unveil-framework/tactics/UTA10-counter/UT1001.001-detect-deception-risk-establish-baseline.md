# UT1001.001 - Detect deception risk: Establish baseline

**Tactic:** [UTA10 - Counter](./README.md)

## Description

The handler may establish the source's (truthfulness) baseline; that is, how they normally act when telling the truth. Indicators must be obtained and registered during every interaction with the source. It must be noted that baselines often are related with the personality profile of the source, hence the need to consider said profile when understanding a baseline, especially since high psychoticism sources find it easier to lie and therefore it is harder to detect them128 . Indicators that would be considered in order to understand the baseline include the following:

Response time

Verbal indicators

Body language

Facial expressions
