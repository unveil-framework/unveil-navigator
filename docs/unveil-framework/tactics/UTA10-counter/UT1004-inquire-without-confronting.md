# UT1004 - Inquire without confronting

**Tactic:** [UTA10 - Counter](./README.md)

## Description

The handler may subtly inquire about the suspicious behavior without directly confronting the source. This approach involves asking open-ended questions or expressing curiosity about the observed actions in a non-threatening manner such as “Sorry. I don't know if I understood you correctly, is this what you mean?” or other similar inquiries. By doing so, the handler can gather additional information and gain insights into the source's motivations while maintaining rapport and avoiding defensive reactions.
