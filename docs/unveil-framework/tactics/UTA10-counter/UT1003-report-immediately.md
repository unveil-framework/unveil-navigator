# UT1003 - Report immediately

**Tactic:** [UTA10 - Counter](./README.md)

## Description

The handler may immediately report any detected indicators of suspicious activity to their superiors or relevant team members. This quick response ensures that potential threats are communicated without delay, allowing the organization to take timely action to mitigate risks.

Prompt reporting helps maintain the integrity and security of the operation by enabling a coordinated response to emerging threats.
