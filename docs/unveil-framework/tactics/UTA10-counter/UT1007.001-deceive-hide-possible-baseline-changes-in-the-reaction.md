# UT1007.001 - Deceive: Hide possible baseline changes in the reaction

**Tactic:** [UTA10 - Counter](./README.md)

## Description

The handler may consciously hide any changes in their behavior or demeanor that might indicate recognition of the source's suspicious activity. By maintaining a consistent baseline in their reactions, the handler prevents the source from realizing that their actions have been detected, thus preserving the element of surprise and control over the situation.
