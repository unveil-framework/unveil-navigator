# UT1007.002 - Deceive: Fake baseline changes

**Tactic:** [UTA10 - Counter](./README.md)

## Description

The handler may deliberately alter their behavior to simulate a change in baseline reactions, creating a false impression for the source. This technique can be used to manipulate the source's perceptions, elicit specific responses or test the source's reactions to perceived changes in the handler's demeanor. It is a strategic way to influence the interaction without revealing true intentions.
