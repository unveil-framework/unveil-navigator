# UT1001.002 - Detect deception risk: Create and enforce a detection plan

**Tactic:** [UTA10 - Counter](./README.md)

## Description

The handler may create a deception detection plan, in which the following phases are conducted in order to detect the source's deceit indicators in a controlled context or environment129: 1. Relax the source, do not ask about the topic used to check for indicators and register the behavioral baseline. 2. Ask a control question, that is, a question of which the answer is known, and check for baseline deviations if they lie. 3. Go back to a relaxation period, change topics and ask unrelated questions. 4. Repeat phases 2 and 3 as long as it is needed. 5. Close the interaction as required.
