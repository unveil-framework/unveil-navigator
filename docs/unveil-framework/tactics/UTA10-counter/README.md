# UTA10 - Counter

## Description

The handler is trying to detect indicators of suspicious activity of the source, as well as to
respond to said indicators in such a way that protects them and the overall operation. The main
difference of this tactic when compared to the previous nine is that, while the rest are proactive,
UTA10 is reactive, meaning that it includes possible ways in which the handler can react to
unexpected behaviors of the source.
This tactic includes what in other TTPs frameworks is dubbed mitigations or “blue” actions, as
it comprises the defensive actions a handler can take when the source is or seems to be trying
to influence, control or obtain anything from the handler.
